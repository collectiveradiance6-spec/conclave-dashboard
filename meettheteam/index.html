The Tech Stack

🤖
Lead Developer
AEGIS v10
Sovereign Edition · Discord Bot & Web Architect

TheConclave Dominion runs on a full custom-built tech stack. The AEGIS v10 "Sovereign Edition" Discord bot handles economy, moderation, Nitrado server monitoring, AI chat (Claude Haiku + Sonnet routing), music systems, ticket management, giveaways, and the full ClaveShard economy.

The website is deployed on Cloudflare Pages with a Supabase backend powering user data, economy records, and server analytics. All 10 map servers are monitored live via Nitrado API integration.
